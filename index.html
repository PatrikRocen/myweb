📌 Jak nahrát web na GitHub Pages

👉 Výsledek: Web poběží na moje-jmeno.github.io/moje-web ZDARMA

1️⃣ Vytvoření účtu na GitHubu

  1. Jdi na GitHub.com
  2. Klikni na Sign Up a vytvoř si účet

2️⃣ Vytvoření nového repozitáře

  1. Klikni na "+" vpravo nahoře → New repository
  2. Zadej název repozitáře (např. moje-web)
  3. Nech ho Public (musí být veřejný pro GitHub Pages)
  4. Zaškrtni "Add a README file"
  5. Klikni na Create repository

3️⃣ Nahrání HTML souborů

Možnost 1: Ručně přes web GitHubu

  1. Otevři svůj nový repozitář
  2. Klikni na "Add file" → "Upload files"
  3. Přetáhni index.html, style.css, obrázky…
  4. Klikni "Commit changes"

Možnost 2: Přes Git (pokročilé, rychlejší pro více souborů)

  1. Stáhni si Git (git-scm.com)
  2. Otevři terminál a napiš:
    sh
    git clone https://github.com/moje-jmeno/moje-web.git
cd moje-web
cp -r /cesta/k/souborum/* .
git add .
git commit -m "První verze webu"
git push origin main

4️⃣ Aktivace GitHub Pages (web bude online!)

  1. V repozitáři jdi do Settings
  2. Nahoře klikni na Pages
  3. V sekci Branch → Main → Save
  4. Hotovo! Tvůj web poběží na https://moje-jmeno.github.io/moje-web/

5️⃣ Připojení vlastní domény (volitelné)

👉 Chceš místo moje-jmeno.github.io vlastní doménu (např. mojeweb.cz)?

  1. Kup doménu (např. Wedos, Forpsi)
  2. Jdi do Settings > Pages > Custom domain
  3. Zadej mojeweb.cz
  4. U svého registrátora změň DNS záznamy (CNAME) podle GitHubu
  5. Hotovo! Tvůj web teď běží na vlastní doméně

html
<!DOCTYPE html>
<html lang="cs">
<head>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Moje webová stránka</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Vítejte na mé stránce!</h1>
        <p>Jednoduchý HTML & CSS web</p>
    </header>

    <section>
        <h2>O mně</h2>
        <p>Ahoj! Toto je moje první webová stránka vytvořená v HTML a CSS.</p>
    </section>

    <section>
        <h2>Kontakt</h2>
        <p>E-mail: <a href="mailto:tvoje@email.cz">tvoje@email.cz</a></p>
    </section>

    <footer>
        <p>&copy; 2024 Moje webová stránka</p>
    </footer>
</body>
</html>
